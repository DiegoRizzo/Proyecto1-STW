<h3>Descripción del Proyecto</h3>
<p>Para este proyecto, desarrollamos una aplicación CRUD con funcionamiento de una red social o blog. El usuario es capaz de leer y crear publicaciones, al igual que realizar modificaciones o eliminar publicaciones por completo de la lista.</p>

<h3>API Utilizada</h3>
<p>La API que se utilizó para este proyecto es DummyJSON, pues esta API brinda recursos gratis y accesibles para trabajar con desarrollo de APIs sin hacer modificaciones a una base de datos real. Esta API nos permite hacer cambios a la interfaz del usuario usando los cuatro verbos HTTP (GET, POST, PUT, DELETE).</p>

<h3>Instrucciones para correr el Proyecto</h3>
<p>No se requiere instalación de dependencias. Clonar el repositorio, entrar a la carpeta del proyecto y abrir <code>index.html</code> con Live Server en VS Code.</p>
